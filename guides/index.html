The Hypercore Protocol

The Hypercore Protocol is a peer-to-peer network for:

  • Connecting devices,
  • Streaming data,
  • Sharing files and databases, and
  • Building applications.

Hypercore is inspired by BitTorrent. It creates networked datasets which users can seed in a distributed mesh. Hypercore includes a variety of tools for connecting devices and is built on a very modular API for building custom programs.

Hypercore currently serves as the foundation for a diverse range of P2P applications, including chat apps (Cabal), mapping software (Mapeo), scientific research sharing (Lib Scie), and even a couple web browsers (Beaker, Agregore). And the whole thing's MIT-licensed, so you can use it as you like!
