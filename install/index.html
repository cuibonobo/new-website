Install the Hyp CLI

Hyp is a command-line tool for Hypercore Protocol. It includes tools for transferring data, seeding files, and running the daemon.

After installing, read the "Using the Hyp CLI" guide to get started.

npm install -g @hyperspace/cli

Requires nodejs 14+

The Hyperspace Daemon

If you're building an application or service, you should look at the Hyperspace Daemon. It provides a complete out-of-the-box toolset for the Hypercore Protocol.

You can also use the individual modules to build a solution that fits your usecase.
