Hypercore Protocol

The peer-to-peer network

Install Guides

Share from any device

Hypercore securely shares files and databases. Like BitTorrent, as more people "seed" a dataset it will download faster. Learn more.

End-to-end privacy

Encrypted connections between devices ensure that you only share with the people you intend to reach.

Built for developers

A suite of Javascript APIs make it easy to build new peer-to-peer applications. Learn more.

// connect to the daemon const client = new HyperspaceClient() await client.ready() // create a hyperdrive p2p folder const drive = hyperdrive(client.corestore()) await drive.promises.ready() await drive.promises.writeFile('/hello.txt', 'Hello World!') // announce to the network client.network.configure(drive.discoveryKey, { lookup: true, announce: true })
